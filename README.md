# Artify Server

A robust Express.js backend server for the Artify platform, providing RESTful API endpoints for artwork management, user authentication, and social features.

## live link : https://artify-sami.netlify.app

## Key Features

• **Artwork Management**: Complete CRUD operations for artworks with support for public/private visibility, search by title or artist name, category filtering, and featured artwork retrieval

• **Firebase Authentication**: Secure user authentication and authorization using Firebase Admin SDK with JWT token verification for protected routes

• **MongoDB Integration**: Efficient data persistence using MongoDB with connection pooling and optimized queries for artworks, favorites, and user data

• **Social Features**: Like/unlike functionality for artworks and a favorites system that allows users to save and manage their preferred artworks

• **Vercel-Ready Deployment**: Optimized for serverless deployment on Vercel with conditional server initialization, supporting both local development and cloud hosting environments
